API

Animamos a desarrollar aplicaciones y otros servicios integrando nuestros ratings en sus trabajos.Esta pagina deberia ayudarte a usar nuestros datos para tu propia aplicacion. Nuestro codigo fuente esta en Github.

Todos los datos en los legibles JSON

El formato de datos es especificado en nuestra wiki. Por favor comprueba la version de la especificacion con la version de la API.

  • Cada servicio (e.g. facebook) tiene un archivo JSON, Miralos todos en https://tosdr.org/api/1/service/

Uso

Nuestro (un)hoster 5apps tiene habilitado CORS. Ejemplo con JQuery para obtener Facebook: 

$.getJSON("https://tosdr.org/api/1/service/facebook.json").done(function(data) {
  return console.log(JSON.stringify(data));
});

Todos los archivos JSON estan publicados bajo la licencia Creative Commons.
